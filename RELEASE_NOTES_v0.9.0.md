# EAS v0.9.0 — AI Assistant Beta

## 추가 기능
- 상품 사진 AI 분석 버튼 및 결과 카드
- 브랜드, 상품명, 카테고리, 색상, 상태, 신뢰도, 검색어 응답 지원
- AI 제안값을 빈 브랜드/상품명과 카테고리/색상/검색어 필드에 반영
- 새상품 조건을 적용한 eBay 검색 바로가기
- 상품 상세 화면에 AI 분석 요약과 신뢰도 표시
- 설정 화면에 안전한 AI 분석 서버 주소 입력란 추가

## 보안 구조
GitHub Pages에는 AI API 비밀키를 저장하지 않습니다. 앱은 사용자가 설정한 HTTPS 서버리스 엔드포인트에 사진과 상품 정보를 보내고 JSON 결과만 받습니다.

## 검증
- app.js 문법 검사
- 서비스워커 캐시 v0.9.0 갱신
- ZIP 무결성 검사

## 실기기 확인 필요
실제 AI 분석은 호환 서버리스 엔드포인트가 연결된 뒤 iPhone에서 확인해야 합니다.
