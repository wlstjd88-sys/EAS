# EAS v0.6.1

iPhone-first eBay sourcing assistant PWA.

## 포함 기능
- 상품 사진 촬영/선택
- 브랜드, 상품명, 바코드 입력
- 매입가(KRW), 예상 판매가(USD) 입력
- 예상 순이익, ROI, 마진율, 손익분기 판매가 계산
- Buy Score 및 BUY / CONSIDER / PASS 판정
- 상품 저장, 편집, 삭제, 검색
- 대시보드 및 빠른 계산
- LocalStorage 데이터 보존
- GitHub Pages 및 홈 화면 설치 지원

## GitHub Pages 배포
이 ZIP의 압축을 푼 뒤 안의 파일과 폴더를 EAS 저장소의 루트에 업로드합니다.
그 다음 저장소 Settings → Pages → Deploy from a branch → main / (root) → Save를 선택합니다.
